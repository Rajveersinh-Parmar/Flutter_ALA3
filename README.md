# Flutter ALA-3: Local Data Storage App

## 📌 Project Overview
This project is developed as part of the GMIU FAD ALA-3 assignment. The application demonstrates the implementation of **local data storage in Flutter using SQLite**.

The app allows users to:
- Add data
- View stored data
- Update existing data
- Delete data

---

## 🚀 Features
- Local database using SQLite
- CRUD operations (Create, Read, Update, Delete)
- Simple and user-friendly UI
- Efficient data handling

---

## 🛠️ Technologies Used
- Flutter (Dart)
- SQLite (`sqflite` package)
- Path package

---

## 📱 How It Works
1. User enters data in input field
2. Data is stored in SQLite database
3. Stored data is displayed in a list
4. User can update or delete any record

---

## 🧪 Output Description
- Data is saved locally on the device
- No internet connection required
- Real-time updates in UI after operations
